### :newspaper: 주요 기능
##### 🙍‍♂️비회원
```
매거진(디테일) 보기
추천장소 지도(kakaomap sdk) 보기
회원가입
```
##### 🙋‍♂️회원
```
좋아요한 장소 모아보기(마이페이지)
일정 추가하기, 일정 보기(v-calendar)
```
##### 부가기능
```
firebase realtime database를 통한 회원관리
github action&firebase를 연동한 CI/CD
```
